#### 1 Write a shell script to check whether the entered number is prime or not

```bash copy
#!/bin/bash

# Get the input number
read -p "Enter a number: " num

# Check if the number is less than 2
if [ $num -lt 2 ]; then
  echo "Invalid input. Number must be greater than 1."
  exit 1
fi

# Check if the number is prime
for ((i=2; i<=num/2; i++))
do
  if [ $((num%i)) -eq 0 ]; then
    echo "$num is not a prime number."
    exit 0
  fi
done

echo "$num is a prime number."

```

#### 2 Write a shell script to calculate HRA of employees depending upon their basic

```bash copy
#!/bin/bash

# Get the basic salary
read -p "Enter the basic salary: " basic

# Calculate the HRA
if [ $basic -lt 5000 ]
then
  hra=`echo "$basic*0.1" | bc`
elif [ $basic -lt 10000 ]
then
  hra=`echo "$basic*0.2" | bc`
else
  hra=`echo "$basic*0.3" | bc`
fi

# Print the HRA
echo "The HRA is: $hra"

```

#### 4 Write a shell script, which takes a filename as command line argument, asks the user if he wants to revoke the read, write permissions for the group and others for that particular file. If the answer is â€œyâ€ then it should do so or else it should abort the operation.

```bash copy
#!/bin/bash

# Get the input file name
file=$1

# Check if the file exists
if [ ! -f $file ]; then
    echo "File not found."
    exit 1
fi

# Ask user if they want to revoke permissions
read -p "Do you want to revoke read, write permissions for the group and others for the file '$file'? (y/n) " choice

# Check user choice
if [ $choice == "y" ]; then
    # Revoke permissions
    chmod o-rw $file
    chmod g-rw $file
    echo "Permissions revoked for the file '$file'."
else
    echo "Aborting operation."
fi

```
