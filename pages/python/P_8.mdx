### 3 ‚¶Å	Write a program to overload ** (exponential) operator.
```py filename="P7_3.py"   copy
class exp:
    def __init__(self,x=0,y=0):
        self.x=x
        self.y=y
    def __str__(self):
        return('{} {}'.format(self.x,self.y))
    def __pow__(self,other):
        x=self.x**other.x
        y=self.y**other.y
        return exp(x,y)
e1=exp(2,3)
e2=exp(2,3)
print(e1.__pow__(e2))
```
