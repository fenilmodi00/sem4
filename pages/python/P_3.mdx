### 1 Write a program to check if number is Armstrong.
```py copy
num = int(input("Enter a number: "))
originalNum = num
rem = 0
modulo = 0
while num != 0:
    modulo = num % 10 
    rem = rem + pow(modulo, 3)
    num = int(num / 10)
if originalNum == rem:
    print(f"{originalNum} is an armstrong number")
else:
    print(f"{originalNum} is not an armstrong number")
```

### 7.	Write a program to count number of digits, upper-case characters and lower-case characters from the string.
```py copy
data="Hello ... How are you? 123" #data=input("Enter string: ")
upper=0
lower=0
special=0
digit=0

for i in data:
    if(i>='A' and i<='Z'):
        upper+=1
    elif(i>='a' and i<='z'):
        lower+=1
    elif(i>='0' and i<='9'):
        digit+=1
    else:
        special+=1
print("Data=",data)
print("Uppercase={}\nLowercase={}\nDigit={}\nSpecial={}".format(upper,lower,digit,special))

```
