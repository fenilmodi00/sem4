### 6 Write a python program to for library charges a fine for books returned late. Following are the fines: 
#### First five days: 40 paisa per day.
#### Six to ten day: 65 paisa per day.
#### Above ten days: 80 paisa per day
```py filename="P2_6.py"   copy
# Get number of days late
days_late = int(input("Enter number of days late: "))

# Initialize fine to 0
fine = 0

if days_late > 0:
    if days_late <= 5:
        fine = days_late * 0.40
    elif days_late <= 10:
        fine = (5 * 0.40) + ((days_late - 5) * 0.65)
    else:
        fine = (5 * 0.40) + (5 * 0.65) + ((days_late - 10) * 0.80)
    print("Fine: Rs.", fine)
else:
    print("No fine")

```

### 7 Write a python program to count odd numbers from given three numbers and display maximum odd number.
```py filename="P2_7.py"   copy
# Get input from user
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
num3 = int(input("Enter third number: "))

# Initialize count and max_odd to 0
count = 0
max_odd = 0

# Check if first number is odd
if num1 % 2 != 0:
    count += 1
    max_odd = num1

# Check if second number is odd
if num2 % 2 != 0:
    count += 1
    if num2 > max_odd:
        max_odd = num2

# Check if third number is odd
if num3 % 2 != 0:
    count += 1
    if num3 > max_odd:
        max_odd = num3

# Display the count and maximum odd number
print("Count of odd numbers:", count)
print("Maximum odd number:", max_odd)
```
